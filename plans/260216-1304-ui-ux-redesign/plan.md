**Target**: `apps/client/`

**Status**: Ready for Implementation

---

## Overview

Transform Pink Nail Salon client website from basic service site to high-end nail art studio digital experience. Core focus: visual polish, masonry gallery with filtering, booking flow enhancement, premium brand aesthetic.

**Key Deliverables**:
- Design System: Dusty Rose (#D1948B) + Cream (#FDF8F5) palette, Playfair Display + Be Vietnam Pro fonts
- Masonry Gallery: Pinterest-style layout with filtering (nail shapes + styles)
- Modal Booking Integration: Gallery modal with "Book this design" CTA
- Booking Flow: Polished 2-step flow, optional 3-step expansion
- Confetti Animation: Celebration on booking confirmation
- Performance: Lighthouse 95+ score, mobile-responsive
